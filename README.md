# Java基础概念

### 常用cmd命令

| 命令            | 说明                 | 举例               |
| --------------- | -------------------- | ------------------ |
| 盘符名称 + 冒号 | 盘符切换             | E：回车，切换到E盘 |
| dir             | 查看当前路径下的内容 |                    |
| cd 目录         | 进入单级目录         | cd mzh             |
| cd ..           | 回退到上一级         |                    |
| cd 目录1/目录2  | 进入多级目录         | cd Steam\PUBG      |
| cd \            | 回退到盘符目录       |                    |
| cls             | 清屏                 |                    |
| exit            | 推出命令提示符窗口   |                    |
